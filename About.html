Countdown Timer

This is a website created by khalid zakariya, made easy as it is a one place to get your time awareness right.

Instruction

Select a date from the calendar and it will appear in the countdown showing how long there is left before the selected time.

NOTE!

If you select a date before the current time you will surely get negative values
